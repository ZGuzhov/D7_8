Проект собран на основе модулей D3-D7. Проект выполняет функцию библиотеки книг. В нём можно просмотреть список книг, добавлять/редактировать/удалять книги. Так же можно добавлять/редактировать/удалять издательства, авторов и друзей.
Реализован функционал работы с пользователями, как локально, так и с помощью GitHub. Для изучения способов работы с шаблонами и формами переоформлены формы логина и выхода. Весь функционал работы с пользователями реализован только на главной странице, так как не видел смысла как-то оформлять и так кривой сайт из кучи заданий.

1. Установите необходимые пакеты:
pip install -r requirements.txt

2. Запустите локальный сервер:
python manage.py runserver

Для входа в админку используйте путь /admin/, логин: admin, пароль: 12345.

Создавать/редактировать/удалять авторов, издательства, друзей и книги можно через админку.
Так же частичный функционал создания доступен прямо на сайте.
